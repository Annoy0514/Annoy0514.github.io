总结

今日学习了使用vue对前后端分离项目，进行一个前后端数据交互。

在后端的 controller 开发完毕的基础上，前端调用方法，查询数据库中的数据，最终对数据进行一个增删改查的操作。

主要需要反复练习的是，在 前端页面怎么调用后端的接口。

大致分为 **四步走**：

(1) 添加路由。—router

(2) 添加点击某个路由，跳转页面的路径

(3) api 文件夹下创建 js 文件，用于编写接口和参数 —api

(4) 创建 vue 页面引入 js 文件，调用方法实现功能 —views
